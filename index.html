Have a station log or historical export? Upload a JSON or CSV file and the almanac will rebuild every plate from your numbers — no network needed.

CSV — one row per day, header required. Values read as metric (°C, mm):

date,tmax,tmin,precip
2026-06-15,41.2,27.8,0.0
